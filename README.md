# psychic-octo-spork
A Python Kafka toolkit intended for transfering serial data from an arduino.

# Requirements:
```
Python 3
Python SMBus
Python Kafka
Python Serial
Python asyncio
Python concurrent
```

# How to configure "ustconfig":
```
kafka_hostname = 'ip or host address'
kafka_port = 'port number'
kafka_connection = kafka_hostname + ":" + kafka_port
```
